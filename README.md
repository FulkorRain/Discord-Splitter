-- DiscordSplitter -- 

Easy to use high-speed, lossless file convertion engine, splitter & merger designed for Discord or any other social media file size limitations.

Converts any file(s) into .bin files, with an optional splitting of large files into parts with customizable sizes to be uploaded properly on Discord or alike.
Preserving full folder structure and able to perfectly merge everything back together.

HOW TO USE:

- Create a folder and place the program inside. Upon first use it'll create four folders. "Input" , "Output" , "MergeInput" , "MergeOutput".

CONVERT:
@ Input folder: Place any file and folder here to convert them into .bin files, and optionally split them.
@ Output folder: All converted and split files/folders from the Input folder are automatically placed here.

MERGE:
@ MergeInput folder: Place any file and folder here to merge them back into their original file forms. Combining any parts if any are found.
@ MergeOutput folder: All merged files/folders are automatically placed here.

- Once opened the program shows different choices, using Options you can disable/enable the FileSplitter feature and customize the size.
- Convert option will turn all files inside the Input folder into .bin files and optionally split them, placing everything into the Output folder.
- Merge option will return all .bin files inside the MergeInput folder. into their original form, and combines parts if found. Placing everything into the MergeOutput folder.
